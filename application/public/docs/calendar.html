Introduction

Get started with Calendar web component which is framework agnostic and can be integrated with any javascript framework.

About

sm-calendar is beautiful web component written in native javascript and it can be very well integrated with any javascript framework.

This calendar web component is very well tested with below frameworks:

  1. Native Javascript apps using Jquery
  2. React JS
  3. Angular JS
  4. Vue JS
